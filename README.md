# Competitive Programming Repository

This repository contains my Competitive Programming journey, including contests, problem-solving practice, topic-based learning, and ECPC preparation.

## Repository Structure

### 🏁 Contests
Solutions and notes from competitive programming contests.

Mainly includes:
- Codeforces Contests
- Virtual Contests
- Contest upsolving

The goal is to track contest performance and learn from post-contest analysis.

---

### 📚 Levels
Problems organized according to learning levels.

Examples:
- Level 0
- Level 1
- Level 2

Each level focuses on mastering a set of concepts and problem-solving techniques before progressing to the next stage.

---

### 🎯 Problems
Problems categorized by difficulty.

Examples:
- Div 2 A
- Div 2 B
- Div 2 C1
- Div 2 C2

This organization makes it easier to practice problems of a specific difficulty level.

---

### 🏆 Gym
Practice sessions and solutions from programming gyms.

Mainly used for:
- ECPC Preparation
- Team Training
- Upsolving and Practice

---

### 🧠 Topic-Based
Problems and resources organized by advanced topics.

Examples:
- Graph Theory
- Dynamic Programming
- Number Theory
- Data Structures
- Greedy Algorithms

This section is dedicated to focused learning and mastering individual topics.

---

## Purpose

- Improve problem-solving skills.
- Build a strong foundation in algorithms and data structures.
- Prepare for competitive programming contests.
- Track learning progress in an organized way.
- Prepare for ECPC and other programming competitions.

---

*"The goal is not just to solve problems, but to continuously reach problems that I cannot solve yet and learn how to solve them."*
